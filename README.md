
# WAWYC - Cyberware Bio Site

A cyberpunk/cyberware themed bio site for wearewatchingyouclosely.net

## Features
- Neon, animated, dark cyberware style
- Social links:
	- [Instagram](https://instagram.com/wearewatchingyouclosely)
	- [Twitter/X](https://x.com/_WAWYC_)
	- [YouTube](https://www.youtube.com/channel/UCbVpO0A_-Rei9DOWCFpxO3Q)
	- [GitHub](https://github.com/wearewatchingyouclosely)
- Built with Vite + React

## Getting Started

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

Customize the theme and content in `src/`.
